Dialog

Allows the creation of UI blocking dialog boxes

Methods

dialog

Opens a dialog with a specific type, text and title.

Returns a promise to be fullfilled when the dialog has been closed.

Erebus.dialog(Erebus.dialog.TYPE.info, 'Message', 'Optional title').then(function() {
    // code executed when the dialog has been closed
}); See running example

dialog.info

Opens an informational dialog with a message and optional title.

Returns a promise to be fullfilled when the dialog has been closed.

Erebus.dialog.info('Message', 'Optional title').then(function() {
    // code executed when the dialog has been closed
});

dialog.success

Opens a success dialog with a message and optional title.

Returns a promise to be fullfilled when the dialog has been closed.

Erebus.dialog.success('Message', 'Optional title').then(function() {
    // code executed when the dialog has been closed
});

dialog.error

Opens an error dialog with a message and optional title.

Returns a promise to be fullfilled when the dialog has been closed.

Erebus.dialog.error('Message', 'Optional title').then(function() {
    // code executed when the dialog has been closed
});

dialog.confirm

Opens a dialog with a message and optional title and collects an user selection.

Returns a promise to be fullfilled when the dialog has been closed and receives the selection.

Erebus.dialog.confirm('Message', 'Optional title').then(function(result) {
    // code executed when the dialog has been closed
});
