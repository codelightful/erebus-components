Toast

Allows the creation of emerging messages

Methods

toast

Opens a toast with a specific type, text and title.

Returns a promise fullfilled with the toast instance.

Erebus.toast(Erebus.toast.TYPE.info, 'Message', 'Optional title').then(function(toast) {
    // code executed when the toast has been opened
}); See running example
